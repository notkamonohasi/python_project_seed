# PYTHON PROJECT SEED

## 使い方(途中から)

1. pip install uv
2. uv sync
3. . .venv/bin/activate

## 使い方(最初から)

1. pip install uv
2. uv init
3. uv python pin ${PY_VERSION}
4. uv add ${PY_PACKAGE}
5. . .venv/bin/activate

## 必要な VSCode 拡張機能

- Mypy Type Checker
- Prettier Code formatter
- Ruff
- shell format
- even-better-toml
